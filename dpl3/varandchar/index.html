DPL3 Variables and Handy Characters

Reference for editors using DPL3

DPL3 Variables

Variable Description
%NR%Sequence number of the item in the list
%PAGE%Full page name including namespace
%PAGEID%Internal numeric page ID
%IMAGE%Image file path used in the page
%PAGESEL%Page matching selection criterion
%IMAGESEL%Image name used in selection criterion
%TITLE%Title without namespace
%NAMESPACE%Page's namespace
%SIZE%Page size in bytes
%SIZEFS%Font-size scaled by page size
%COUNT%Page view count (deprecated)
%COUNTFS%Font-size by log usage count (deprecated)
%COUNTFS2%Font-size by sqrt usage count (deprecated)
%DATE%Date from query (e.g. edit date)
%USER%Last editor username
%CONTRIBUTOR%User who made the edit
%CONTRIBUTION%Byte difference in edit
%CONTRIB%Asterisks proportional to contribution
%CATLIST%Pipe-separated list of categories
%CATBULLETS%Bulleted list of categories
%CATNAMES%Comma-separated category names
%REVISION%Revision ID of the page
%EDITSUMMARY%Edit summary/comment
%EXTERNALLINK%URL from external link query
%PAGES%Total returned pages (header only)
%TOTALPAGES%Total pages ignoring count= limit
%VERSION%DPL3 version number
%DPLTIME%DPL generation time
%FIRSTNAMESPACE%Namespace of first result
%FIRSTTITLE%Title of first result
%LASTNAMESPACE%Namespace of last result
%LASTTITLE%Title of last result
%SCROLLDIR%Scroll direction (from URL)
