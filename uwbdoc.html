Utaite Wiki Bot Commands

The Utaite Wiki Bot provides several commands to help you interact with the Utaite Wiki. Below you'll find detailed documentation for each command.

!searchcat

Description

Search for pages in one or more categories with paginated results.

Usage

!searchcat category_name
!searchcat category1, category2, category3

Features

  • Shows clickable links to wiki pages
  • Pagination with arrow buttons (⬅️ and ➡️)
  • Go to specific page functionality
  • Displays 10 results per page
  • Shows total result count

Examples

!searchcat Male Utaite

!searchcat Male Utaite, Deceased

!template

Description

Explains a template's usage and parameters with detailed documentation.

Usage

!template template_name

Features

  • Shows template usage syntax
  • Displays comprehensive documentation
  • Handles long documentation with multiple embeds
  • Removes HTML tags for cleaner display

Examples

!template Song_Infobox

!template Utaite_Infobox

!searchfile

Description

Search for a file and retrieve its license information.

Usage

!searchfile filename
!searchfile File:filename

Features

  • Automatically adds 'File:' prefix if not provided
  • Displays file license information
  • Shows error message if file not found

Examples

!searchfile soraru.jpg

!searchfile File:mafumafu.png
