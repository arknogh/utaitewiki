How to Use This Tool

Basic Steps

  1. Paste your old format template in the left box.
  2. Click "Convert Template".
  3. Review the parsed data in the table that appears.
  4. Copy the new format result from the right box.

CD Field Handling

  • If content exists before # in CD field (e.g., EXIT TUNES Albums#Ikemen Voice Paradise 7), that value replaces the singer.
  • If no content before # (e.g., #Hakozume Clover), original singer value is kept.
  • Album name (after #) is always appended to the singer value (e.g. `singer=SingerName#AlbumName`).
  • An `alias` field is generated (plain text) if an album is specified.

Keyboard Shortcuts

  • Ctrl + Enter Convert template
  • Ctrl + Alt + C Copy output
  • Esc Clear fields
